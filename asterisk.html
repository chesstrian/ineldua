Instalación de Asterisk

Prerrequisitos

  • Instalación Sistema Operativo Debian 6.
  • Actualización del Sistema Operativo: 
                  % apt-get -y install sudo
                
                  $ sudo apt-get update
                  $ sudo apt-get upgrade

Paquetes Fuente

                  $ cd /usr/src/
                  $ wget http://downloads.xiph.org/releases/speex/speex-1.2rc1.tar.gz
                  $ wget http://downloads.asterisk.org/pub/telephony/dahdi-linux-complete/dahdi-linux-complete-current.tar.gz
                  $ wget http://downloads.asterisk.org/pub/telephony/certified-asterisk/certified-asterisk-1.8.15-current.tar.gz
                  $ wget http://mirror.freepbx.org/freepbx-2.9.0.tar.gz

Descompresión

                  $ mkdir tar
                  $ for i in `ls *gz`
                  > do
                  >   tar xf $i
                  >   mv $i tar/
                  > done

Dependencias Necesarias

                  $ cd /usr/src/certified-asterisk-1.8.15-cert2/contrib/scripts/
                  $ sudo ./install_prereq install
                  $ sudo apt-get -y install gcc g++ make libncurses5-dev apache2 libxml2-dev mysql-server mpg123 php5 php-pear  \
                  php5-mysql libmysqlclient15-dev linux-headers-`uname -r` php5-gd libmyodbc postfix libtiff4-dev libtiff-tools \
                  imagemagick gs
                  $ sudo pear install DB

Compilación e Instalación

Speex

                  $ cd /usr/src/speex-1.2rc1/
                  $ sudo ./configure
                  $ sudo make && sudo make install

DAHDI

                  $ cd /usr/src/dahdi-linux-complete-2.6.2+2.6.2/
                  $ sudo make clean
                  $ sudo make all
                  $ sudo make install
                  $ sudo make config

Asterisk

                  $ cd /usr/src/certified-asterisk-1.8.15-cert2/
                  $ sudo make clean
                  $ sudo ./configure
                  $ sudo make && sudo make install
                  $ sudo make samples

Usuario asterisk

Permisos servicio Asterisk

                  $ sudo useradd -r -c "Asterisk PBX" -d /var/lib/asterisk asterisk
                  $ sudo chown -R asterisk /var/run/asterisk
                  $ sudo chown -R asterisk /var/log/asterisk
                  $ sudo chown -R asterisk /var/lib/asterisk/moh
                  $ sudo ln -s /var/lib/asterisk/moh /var/lib/asterisk/mohmp3

Permisos servicio Apache

                  $ sudo sed -i 's/www-data/asterisk/g' /etc/apache2/envvars
                  $ sudo chown -R asterisk /var/lock/apache2
                  $ sudo /etc/init.d/apache2 restart

Instalación de FreePBX

                  $ cd /usr/src/freepbx-2.9.0/
                  $ mysqladmin -pPassword create asterisk
                  $ mysqladmin -pPassword create asteriskcdrdb
                  $ mysql -pPassword asterisk < SQL/newinstall.sql
                  $ mysql -pPassword asteriskcdrdb < SQL/cdr_mysql_table.sql
                  $ mysql -pPassword ## Entra a MySQL
                
                  mysql> GRANT ALL PRIVILEGES ON asteriskcdrdb.* TO asteriskuser@localhost IDENTIFIED BY 'password';
                  mysql> GRANT ALL PRIVILEGES ON asterisk.* TO asteriskuser@localhost IDENTIFIED BY 'password';
                  mysql> FLUSH PRIVILEGES;
                
                  $ cd /usr/src/freepbx-2.9.0/
                  $ sudo safe_asterisk
                  $ sudo ./install_amp
